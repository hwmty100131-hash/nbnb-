# مشروع مراقبة الأوامر ورفعها إلى GitHub

هذا المشروع مكتوب بلغة بايثون لمراقبة الأوامر القادمة من:
- شبكة الواي فاي
- البلوتوث
- إشارات كهربائية (إن أمكن)

ويقوم برفع البيانات تلقائياً إلى مستودع GitHub.

## المتطلبات
- Python 3.14+
- مكتبات: socket, pybluez, gitpython

## طريقة العمل
1. يلتقط البرنامج الأوامر من المصادر المذكورة.
2. يخزنها محلياً.
3. يرفعها تلقائياً إلى GitHub.

## ملاحظات
- إشارات الكهرباء تتطلب عتاد خاص أو واجهة GPIO (غالباً غير متوفرة على الكمبيوتر العادي).
- يجب إعداد توكن GitHub واسم المستودع مسبقاً.

## للتنفيذ
```bash
pip install -r requirements.txt
python main.py
```
